  • Imagens dinamica

    • Foto dinamica é uma mesma foto de tamanhos diferente para cada aparelho,
    primeio adicina a foto mais pequena usando souce 

      🔴🟡🟢
                    
1 <body>    
2    <souce media="(max width: 750px)" srcset="a imagem" type="imagem/tipo da imagem">
3 <body>

    esse 750px é o tamanho da tela então quando a tela for menos que 750 pixel vai ser mostrado essa imagem é importante falar que a ordem é da menos para a maior

    Exemplo de codigo para tres telas

      🔴🟡🟢

1 <body>    
2    <souce media="(was width: 750px)" srcet="imagens/foto-p.png" type="imagem/png>
3    <souce media="(was width: 1000px)" srcet="imagens/foto-m.png" type="imagem/png>
4    <img src="" alt="">
5 <body>

    Esse codigo se a tela for menor que 750 pixels vai ser exebido a imagem p se for menor que 1000 e maior que 750 vai ser exibida a imagem m e se for maior que 1000 vai ser exebido a imagem g.

hiper textos

  • Criar hiper texto

    • Para criar um hiper texto que é um texto ou uma palavra que direciona para outra pagina usa a tag a,envelope a frase, texto ou palavra que vai virar um hiper texto com a tag a que altomaticamente vai gerar o href que é onde voçe vai colocar a URL, quando os user clicar no texto eles vai ser direcionado para a pagina da URL no href.

      🔴🟡🟢

1 <body>    
2   <a href="url da pagina" target="_blank" rel="external">o que estiver aqui vai virar um hiper texto</a>
3 <body>
  • Pagina interna

    • para cria um hipe texto para uma pagina externa vai no
    href="" coloca o cusor entres as "" e aperta CTRL + ESPAÇO e seleciona a pagina
    OBS: a pagina tem que ta na mesma pasta do arquivo index.

  • abrir em outra guia

    • Para abrir em outra guia é só criar um target="_blank"

  • abrir na na mesma guia

    • target="_self"

  • sem aval

    • rel="nofollow"
